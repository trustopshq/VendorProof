# FAQ

## Do I need to run the import script?

No. It is optional and intended for technical users. Manual setup works without it.

## Does this create databases for me?

No. This helper only imports sample data into the databases created by the Marketplace template.

## Do I need to create a Notion integration?

Yes, to import data via the Notion API. You can skip this if you use manual setup.

## Is this a compliance guarantee?

No. This is a workflow and tracking tool only.
