# Troubleshooting

## Python not found

Install Python 3.10+ and ensure it is added to PATH.

## Data source not found

- Make sure the Marketplace template is installed.
- Share the database with your integration.
- Ensure database names are unchanged: Vendors, Assessments, Question Library, Assessment Items.

## 403 or permission errors

- Integration must have insert content capability.
- Confirm the database is shared with the integration.

## CSV import issues

- Ensure you are using the provided CSV files.
- Do not edit column headers.
